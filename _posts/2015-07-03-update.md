##動画トリミング
ffmpeg -i 入力ファイル -ss 開始秒数 -t 出力ファイル

##動画から静止画への切り出し
ffmpeg -i 入力ファイル -r FPS指定 出力ファイル

##静止画から動画へ結合
ffmpeg -i FPS指定 -i 入力ファイル -vcodec mjpeg gscale 0 出力ファイル

##画面サイズ変更，アスペクト比変更
ffmpeg -i 入力ファイル -s サイズ -aspect アスペクト比 -f コーデック名 出力ファイル
